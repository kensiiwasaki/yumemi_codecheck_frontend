# 都道府県別人口推移グラフ
## 環境構築
### リポジトリのクローンを生成
```
git clone https://github.com/kensiiwasaki/yumemi_codecheck_frontend
```

### パッケージのインストール
```
yarn
```

### .env.localを作成し.env.exampleを元にAPIキーを設定

### 動作確認
```
yarn dev
```

## デプロイ先
```
https://yumemi-codecheck-frontend-zeta.vercel.app/
```
